# Get Connected Parts

## Description

Returns a table of all parts connected to an object.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Part (Object)

You give the path to an object.

* [x] Can use Variable

```
workspace.Baseplate
```

## Outputs

#### PartsTable (Table)

You give the block the variable you want the table of connected parts to be saved to.

```
ConnectedParts
```
