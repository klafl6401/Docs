# Create Object

## Description

Creates an object.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### ClassName (String)

You give the ClassName of an object you want.

* [x] Can use Variable

```
Part
```

#### Parent (Object)

You give the parent of the object you want to create.

* [x] Can use Variable

```
workspace
```

## Outputs

#### Object (Object)

You give the block the variable you want the object to be saved to.

```
NewPart
```
