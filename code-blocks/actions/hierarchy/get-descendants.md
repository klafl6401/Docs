# Get Descendants

## Description

Returns a table of all of an objects descendants.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Parent (Object)

You give the path to an object.

* [x] Can use Variable

```
workspace
```

## Outputs

#### DescendantTable (Table)

You give the block the variable you want the objects table to be saved to.

```
AllObjectsTable
```
