# Get Random Child

## Description

Returns a random child of an object.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Parent (Object)

You give the path to an object.

* [x] Can use Variable

```
workspace
```

## Outputs

#### Result (Object)

You give the block the variable you want the random object to be saved to.

```
RandomObject
```
