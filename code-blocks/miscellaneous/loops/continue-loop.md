# Continue Loop

## Description

Similar to the `continue` keyword in Luau. It would skip the current iteration of the loop and go to the next.
For example, if the index was 1, this block would make it skip to 2.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields
