# Loop Through Descendants

## Description

Iterates through all the descendants of an instance.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Object (Object)

The instance whose descendants will be looped through.

* [x] Can use Variable

```
game.Workspace
```

## Outputs

#### Descendant (Variable)

The current descendant of the instance that is being looped through. 

```
ExampleVariable
```
