# Type Of

## Description

Returns the type of a variable.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Variable (Any)

The variable you want to get the type of.

* [x] Can use Variable

```
SomeVariable
```

## Outputs

#### Type

The type of the variable.

```
number
```
