# If

## Description

Runs connected blocks if the given condition is true.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Value 1 (Any)

The first value you want to compare.

* [x] Can use Variable

```
33
```

#### ComparisonType (Choice)

The type of comparison you want to make.

* [ ] Can use Variable
```
~=
```

#### Value 2 (Any)

The second value you want to compare.

* [x] Can use Variable

```
44
```
