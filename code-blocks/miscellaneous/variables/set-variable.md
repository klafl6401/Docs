# Set Variable

## Description

Sets a variable.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Value (Any)

The thing you want to set the variable to.

* [x] Can use Variable

```
workspace.Baseplate
```

## Outputs

#### VariableName (String)

The string of the name of the variable you set.

```
basePlate
```
