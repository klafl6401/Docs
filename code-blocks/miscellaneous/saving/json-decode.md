# Json Decode

## Description

This block works a lot like `HttpService:JSONDecode(table)`. It turns the given JSON into a table.

#### Available to

* [x] Scripts
* [ ] LocalScripts
* [ ] Yields

## Inputs

#### JSON (String)

The JSON that gets turned into a table.

```
ExampleVariable
```

## Outputs

#### Table (Table)

The table that was decoded from the JSON

```
TableVariable
```
