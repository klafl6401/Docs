# Json Encode

## Description

This block works a lot like `HttpService:JSONEncode(table)`. It turns the given table into a JSON string. 

#### Available to

* [x] Scripts
* [ ] LocalScripts
* [ ] Yields

## Inputs

#### Table (Table)

The table that will be encoded into JSON.

```
Table Variable
```

## Outputs

#### JSON (String)

The JSON from the encoded table as a string.

```
ExampleVariable
```
