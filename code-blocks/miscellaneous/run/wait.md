# Wait

## Description

Similar to the `task.wait` function in Luau.
Waits a certain amount of time before running the blocks connected to it.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [x] Yields

## Inputs

#### Time (Number)

The amount of time the thread will wait before continuing.

* [x] Can use Variable

```
1
```
