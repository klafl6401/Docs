# Render Stepped

## Description

Runs every frame. DeltaTime is the time in seconds since the last frame.

#### Available to

* [ ] Scripts
* [x] LocalScripts
* [x] Yields

## Outputs

#### deltaTime (Number)

The time in seconds since the last frame.

```
deltaTime
```
